# プロジェクト概要

英単語の登録、照会機能を持ったWebアプリケーション。
日々の英語学習の中で出会う意味の分からない英単語を学習記録として単語のスペルとその時に学習した意味を登録し、
学習の復習ができるアプリケーションを作成します。

# 技術スタック
- 言語: TypeScript
- フロントエンド: React
- バックエンド: nestjs
- インフラ: Docker
- DB: MYSQL

# 開発原則
- 対話・ドキュメントはすべて日本語で記述すること
- API は RESTful 設計を基本とする
- テストは必ず実装すること

# ドメイン情報
- 単語（Word）: スペル、意味、例文（任意）、登録日
- 学習状況: 未学習 / 学習中 / 習得済み などのステータス管理
- タグや分類で単語をグルーピングできると望ましい（任意）

# 非機能要件
- 認証機能は必要
- レスポンシブ対応は必要
